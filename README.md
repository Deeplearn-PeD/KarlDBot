# Karl D. Bot
Karl is a coder bot that uses reinforcement learning  to produce high quality code.

## How to Use Karl

Karl the coder bot uses reinforcement learning to produce high-quality code. This tutorial will guide you through the steps to configure and use KarlDBot.

### Prerequisites

Ensure you have the following installed:
- Python 3.12 or higher
- `pip` (Python package installer)
- `PyYAML` library
- `fire` library

You can install the required libraries using the following commands:

```sh
pip install pyyaml fire
```

### Configuration

Create a configuration file named `problem.yaml` in the root directory of the project. This file should contain the details of the data science problem you want to solve. Below is an example configuration:

```yaml
problem_name: Climate analysis
data_source: data.csv
description: Analysis of climate data to identify trends and patterns.
llm_model: gpt-4o
```
### Usage
KarlDBot can be controlled via the command line using the cli.py module. The following commands are available:
To train the agent to solve the problem specified in the yaml file, use the following command:

```sh
```Bash
karl train --config problem.yaml
```

### viewing the report
To view the report generated by the agent after training, use the following command:

```sh
karl view_report --config problem.yaml
```
